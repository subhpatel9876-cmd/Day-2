 1.  CPU Processor Socket:

Yahan CPU (processor) fit hota hai. Yeh motherboard ka sabse important part hai. Bina CPU ke computer kuch bhi process nahi kar sakta.

2.  RAM Memory Slots:
Yahan RAM sticks lagti hain. RAM temporary memory hai — jab tum koi app open karte ho, uska data RAM mein store hota hai taaki CPU fast access kar sake.

3 . 8-pin Power Connector
Sirf CPU ko extra power dene ke liye. Bade processors ko zyada current chahiye hoti hai.

4.  SATA Ports
HDD aur SSD yahan connect hote hain cable ke through. Ek motherboard mein 4-6 SATA ports hote hain.

5.  PCI Slots
Purane zamane ke expansion cards yahan lagte the — jaise sound card, network card. Aajkal PCIe use hota hai.

6.  Audio Port
Headphone aur speaker yahan connect hote hain. Green = speaker, Pink = mic, Blue = line-in.
